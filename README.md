# Social-Media

This is a social media application created using React (v 17.0.2) and NodeJS

**Functionalities:**
1. Register
2. Login
3. Upload a post
4. Like/Dislike post
5. Follow/Unfollow people

**Framework/Libraries:**
1.React 
2.Axios
3.Timeago
4.Dotenv
5.Helmet
6.Multer
7.Morgan
